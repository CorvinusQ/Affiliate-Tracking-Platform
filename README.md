# Affiliate Tracker QA

Тестовый проект для демонстрации навыков тестирования веб-приложений и API.

## 📋 О проекте

Платформа для арбитража трафика — инструмент для маркетологов, которые работают с рекламными кампаниями. Проект включает:
- Веб-интерфейс для управления кампаниями
- REST API для интеграции с внешними источниками
- Систему аналитики и отчетности

## 🎯 Что протестировано

### Функциональное тестирование
- Авторизация и управление сессиями
- Создание и редактирование рекламных кампаний
- Фильтрация и экспорт отчетов
- Интеграции с внешними API (Facebook, Google Ads)

### API тестирование
- Валидация эндпоинтов REST API
- Проверка статус-кодов и схем ответов
- Тестирование обработки ошибок
- Проверка аутентификации и авторизации

### Автоматизация
- E2E тесты на Python + Playwright
- API тесты на Python + requests
- Регрессионные сценарии

## 🛠 Технологии

- **Python 3.11+**
- **Pytest** — фреймворк для тестирования
- **Playwright** — автоматизация браузера
- **Requests** — HTTP запросы
- **Allure** — отчеты о тестах
